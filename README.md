# Sri Ceylon Porcelain Store

Modern e-commerce storefront built with **Next.js 16**, **TypeScript**, and **Tailwind CSS**.

This repository contains the public-facing website that integrates securely with the internal ERP system via API.

---

## 🚀 Tech Stack

- Next.js 16 (App Router)
- TypeScript
- Tailwind CSS
- React Compiler
- GraphQL (ERP integration)
- Firebase Authentication (planned)
- Hosted on Vercel

---

## 🧱 Project Structure

```
src/
  app/
    layout.tsx
    page.tsx
    products/
      page.tsx
      [slug]/
        page.tsx
```

## 🛠 Local Development

Install dependencies:

```bash
npm install
```

Start development server:

```bash
npm run dev
```

Open:

http://localhost:3000
